🚀 Overview

Luxe Store is a modern, responsive e-commerce frontend UI built using only HTML and CSS.
This project focuses on clean design, layout structure, and user experience without JavaScript or backend.

✨ Features
Fully responsive layout
Modern luxury UI design
Product showcase grid
Navigation bar with sections
Hero banner section
Category-based display
Hover effects on products
Clean typography
Mobile-friendly design
Reusable components
Simple and lightweight
Fast loading
Organized folder structure
Beginner-friendly code
Easy customization
📄 Sections
Section	Description
Home	Landing page with hero banner

🏁 Getting Started
1. Open Directly
Download project
Open index.html in browser
2. Live Server (Best Option)
Install Live Server (VS Code)
Right-click → Open with Live Server
3. GitHub Pages
Push project to GitHub
Go to Settings → Pages
Deploy from main branch
